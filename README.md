## рҹ”җ Authentication *(Frontend Only)*

- **Signup fields:** Name В· Email В· Password
- **Login fields:** Email В· Password
- **Validation:** Valid email format В· Password min 6 characters
- User stored in Redux (`authSlice`) and `localStorage` key: `pinrest_user`
- Logout clears user and redirects to login
- вҡ пёҸ No real server or database

---

## рҹ—ғпёҸ Redux Slices

| Slice | Responsibility |
|---|---|
| `authSlice` | Current user, login, signup, logout |
| `pinsSlice` | Pin list, category filter, toggle save |
| `searchSlice` | Search query for filtering the home feed |

---

## рҹҢҷ Theme

- `ThemeContext` manages light/dark mode
- Preference saved in `localStorage` key: `pinrest_theme`
- Toggle available in navbar via the `ThemeToggle` component

## рҹ—ӮпёҸ Data

Each pin includes:

```
id В· title В· description В· category В· image URL В· creator В· followers В· saved
```

> рҹҢҗ Images use `picsum.photos` placeholder URLs вҖ” **requires internet to load**
