# 🌟 About The Project

Pinrest is a modern Pinterest-inspired frontend web application built using **React.js**, **Redux Toolkit**, and **Vite**.

This project focuses on creating a clean, responsive, interactive, and visually attractive UI similar to Pinterest while practicing:

- React Hooks
- Redux Toolkit
- React Router DOM
- Context API
- CSS Modules
- Responsive Design
- LocalStorage Persistence
- Component Reusability
